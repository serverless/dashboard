# Changelog

All notable changes to this project will be documented in this file. See [standard-version](https://github.com/conventional-changelog/standard-version) for commit guidelines.

## 0.2.1 (2023-06-21)

### Bug Fixes

- Temporarily turn off response observation

## 0.2.0 (2023-06-19)

### ⚠ BREAKING CHANGES

### Features

- Update sampling algorithm to not sample infrequently invoked lambdas and to increase the sampling rate to 20%

## 0.1.17 (2023-05-15)

### Bug Fixes

### Performance Improvements

- Improve size of written telemetry log by gzipping
- Remove unused dependencies
- Trim protobuf library by removing unused implementation
- Remove byte-code from layer

- Fix static type checking issues
